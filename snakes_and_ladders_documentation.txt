
Snakes and Ladders (WebGL Implementation)
=========================================

This JavaScript code renders a Snakes and Ladders board game using WebGL. It creates a 10x10 grid, supports two players with movement via dice rolls, and displays snakes and ladders as colored lines. Tile numbers are rendered using an overlaid HTML layer.

Canvas & WebGL Context Initialization
-------------------------------------
const canvas = document.getElementById('gameCanvas');
const gl = canvas.getContext('webgl');
gl.clearColor(0.8,0.8,0.8,1.0);

- Initializes WebGL rendering context.
- Sets the background color to light gray.

Board & Player Setup
---------------------
const gridSize = 10;
const cellSize = 2 / gridSize;

- gridSize: Number of tiles per row and column.
- cellSize: Size of each cell in normalized device coordinates (NDC).

let players = [
  { pos: 0, color: [1, 0, 0, 1] },
  { pos: 0, color: [0, 0, 1, 1] }
];
let currentPlayer = 0;

- Two players initialized at position 0.
- Each player has a unique RGBA color.

Snakes & Ladders
-----------------
const ladders = { 3: 13, 8: 30, 27: 83, 39: 58 };
const snakes  = { 16: 6, 86: 23, 98: 77, 72: 52 };

- Represented as mappings from starting to ending cell index.

Buffer Creation
----------------
const buffer = gl.createBuffer();
gl.bindBuffer(gl.ARRAY_BUFFER, buffer);

- A single buffer is used for all vertex data.

Shader Program
---------------
Vertex Shader:
attribute vec2 pos;
void main() {
  gl_Position = vec4(pos, 0, 1);
}

Fragment Shader:
precision mediump float;
uniform vec4 col;
void main() {
  gl_FragColor = vec4(col);
}

Compilation and Linking:
- Sets up a basic shader pipeline to color and position squares and lines.

Attributes and Uniforms
------------------------
const positionLoc = gl.getAttribLocation(program, 'pos');
const colorLoc = gl.getUniformLocation(program, 'col');

- Position attribute (pos) and color uniform (col) for drawing.

Tile Coordinates Helper
------------------------
function cellToCoords(cell)
- Converts a board index (0-99) to (x, y) coordinates in NDC.
- Handles zigzag layout of Snakes and Ladders rows.

Drawing Functions
------------------
function drawSquare(x, y, size, color)
- Draws a filled square at the given (x, y) with specified size and color.

function drawLine(x1, y1, x2, y2, color)
- Draws a line between two points with specified color.

Draw Tile Numbers (HTML Layer)
-------------------------------
function drawTileNumbers()
- Uses an overlaid #numberLayer div to show tile numbers using absolute positioning.

Game Board Rendering
---------------------
function drawBoard()
- Clears screen.
- Draws grid lines.
- Renders ladders (green) and snakes (red).
- Draws each player’s square.
- Renders numbers on top.

Dice Roll Animation
--------------------
function animateDiceRoll(callback)
- Animates dice roll visually in 10 rapid steps.
- Calls callback(finalRoll) after animation ends.

function rollDice()
- Moves the current player based on dice roll.
- Applies ladder or snake jump if landing on their base/head.
- Switches to next player and redraws the board.

Initialization
---------------
drawBoard();
- Renders the initial state of the game.

HTML Assumptions
-----------------
- A <canvas id="gameCanvas"> element for WebGL rendering.
- A <div id="numberLayer"> positioned over the canvas to show tile numbers.
- A <div id="diceResult"> to show dice values.
