# WebGL Project

## Overview
A WebGL-based graphics project exploring 2D/3D rendering concepts using JavaScript and the HTML canvas element.

## Features
- WebGL rendering pipeline
- Interactive graphics
- Custom shape rendering
- Basic animation support

## Technologies
- WebGL
- JavaScript
- HTML5 Canvas

## How to Run
1. Open project folder
2. Run index.html in browser

## What I Learned
- GPU rendering pipeline basics
- Shader concepts
- WebGL coordinate systems
